# pyro-displacements

Supporting information for: K. Eklund, A. J. Karttunen, Pyroelectric Effect in Tetragonal Ferroelectrics BaTiO<sub>3</sub> and KNbO<sub>3</sub> Studied with Density Functional Theory.

The repository includes modified `displace.py` and `GenDisplacement.py` scripts that can be used together with the [ALAMODE](https://github.com/ttadano/alamode) software by T. Tadano (version 1.5.0).
